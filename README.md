# Тренажёр устной части ЕГЭ по китайскому языку

Локальный тренажёр по открытому варианту ЕГЭ 2026. В нём есть полный экзаменационный режим, отдельная тренировка заданий 1–3, официальные тайминги, автоматические переходы, запись с микрофона и прослушивание ответов.

## Запуск

Из папки проекта выполните:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`. Для записи разрешите браузеру использовать микрофон.

Переключатель «Быстрый режим» сокращает таймеры и удобен для демонстрации. Записи хранятся только в памяти текущей вкладки и исчезают после её закрытия.
